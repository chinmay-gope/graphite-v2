# Changelog

All notable changes to Graphite will be documented in this file.

The format is based on Keep a Changelog and follows Semantic Versioning.

---

## [2.0.0] - Upcoming

### Added

- New project architecture
- Abstract Graph hierarchy
- DirectedGraph and UndirectedGraph implementations
- Redesigned Builder API
- Property-based Random Graph Generator
- Internal package separation
- Improved benchmarking framework
- New documentation

### Changed

- Package structure completely redesigned
- Algorithms grouped by category
- Validation framework redesigned
- Graph creation API simplified
- Result objects improved

---
